Unit 1 Notes



A network is a set of computers sharing information by network nodes. Computers use communication protocols over digital interconnections to communicate. They are made up of network technologies based upon physically wired, optical, and wireless radio frequency methods. PAN is a personal area network with a short range of a few meters. These typically use technologies such as Bluetooth. LAN is a local area network typically with the range of a hundred or so meters. These networks are almost always ethernet based. CAN refers to a campus area network connecting multiple floors within a large building such as SAIT or U of C. WAN is an extremely large network area such as cities or even countries. These use different technologies such as SONET, Frame Relay, Ethernet-WAN Telephone(PSTN). ISP's your Internet Service Providers link isolated areas together. A switch uses a MAC address while a Router uses an IP address. Routers and switches have the same components as computers but they are customized and doing a specific tasks.
